はじめに

最近，転職したのもあり改めてキャリアについて考える機会が訪れたのと，年齢も30歳を超えてきたので，今後の方向性なども考えていきたいなという気持ちが高まり，少し整理するためにBlogを書いています．思考の過程を書いておくのも後から振り返った時に良いかなという心持ちです． 大そうなタイトルを付けてますが，考えたいことは大きく2つで「過去の自分を知る」・「これからしたいこと/興味があることを考える」になります．

過去の自分を知る

これからどうゆうことをしたいかを考えていく上でも，過去の自分を振り返って知ることは大事だと思います．
